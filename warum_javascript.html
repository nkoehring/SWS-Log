    Warum JavaScript?

    Die Zeiten, in der JavaScript eine gefährliche Frickelei war, die einem das Leben und Browsen schwer macht, sind lange vorbei. Heute ist es zu einem nützlichen Werkzeug geworden, welches die Usability einer Website nennenswert verbessern kann.

    Dieser Blog ist komplett in JavaScript realisiert. Auf Serverseite werden nur statische Inhalte ausgeliefert, aus denen JavaScript all die gewohnte Interaktivität zaubert – in deinem Browser! Das entlastet meinen Server und deine Bandbreite.

    Auf Github kannst du dir den Code dieser Seite anschauen. Jeder, der sich ein wenig mit JavaScript auskennt, kann so schnell feststellen, dass da nichts "böses" dran ist. Und selbst wenn, hat man in modernen Browsern auch längst nichts mehr zu befürchten – denn wie gesagt: die Zeiten sind lange vorbei.
