    Why JavaScript?

    The time, where JavaScript was a dangerous fiddle which can make live and browsing really hard is long gone. Today it has became a valuable Tool, which improves the usability of a webpage considerably.

    This blog is completely realised with JavaScript. Only static content is provided by the server and JavaScript magically generates all the known interactivity of it – in your browser! This causes less stress on my server and your internet connection.

    You can investigate the code of this page on Github. Everyone who knows a little of JavaScript can quickly find out that there is nothing "bad" on it. And even if there is something, nobody needs to be afraid in a modern Browser – as I already said: The time is long gone.
